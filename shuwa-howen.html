How to use sign language AI

When you speak using sign language, AI recognizes the sign language and speaks.
Since the type of greeting you want to use is decided, please try using it after seeing how to use it.

1. First

From the tabs at the top of the site, select the type of sign language you want to use.
Currently, there are six types: "Greetings", "Greetings 2", "Replys", "Feelings", "School"and "New Word".
The words that can be used are also written on each page, but please refer to the "Sign Language List" for how to use them.

2. Start the camera

Press the "Start Camera" button in the upper left corner and wait for a while, then you will see a screen asking you to use the camera.
Select the "right button. After a while, the camera will start up.

3. Make sign language recognized

Press the "Start AI" button in the upper right corner, and the number on the right side will start counting down.
When the number reaches zero, please make a sign language.
If there is an interval of 0.7 seconds between each action, it is easier to recognize.
The recognized sign language will be synthesized into speech.
At the same time, deaf people will be able to see the recognized speech in text as well as in speech.

4. Translate spoken voice into text.

Press the "Speak" button on the right half of the screen to start voice recognition.
When the permission to use the microphone appears, select the "right button".
The words you speak will be displayed in text form.
